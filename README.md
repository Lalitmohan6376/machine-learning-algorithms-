Machine Learning Algorithms 
This repository contains implementations of fundamental Machine Learning algorithms built from scratch and/or using standard Python libraries.
The goal of this project is to understand the core concepts, mathematics, and working principles behind popular ML algorithms.
This repository is useful for:
Beginners learning Machine Learning
Students preparing for interviews
Hands-on practice for ML concepts
Academic and personal projects
Algorithms included in this repository:
Linear Regression
Logistic Regression
Support Vector Machine (SVM)
Naive Bayes
Decision Tree
K-Nearest Neighbors (KNN)
Random Forest
K-Means Clustering
Principal Component Analysis (PCA)
Gradient Descent
etc...
